MaxPV! est un programme pour ESP8266 écrit avec l'IDE Arduino. MaxPV! utilise les bibliothèques Arduino et ESP8266 core pour Arduino.

Les bibliothèques externes suivantes sont également employées :
ESPAsyncWiFiManager
ESPAsyncWebServer
AsyncMQTT_Generic
AsyncElegantOTA
ArduinoJson
TickerScheduler
NTPClient
AsyncPing
SimpleFTPServer

L'interface Web est construite avec Bootstrap.
Les graphiques sont affichés avec Highcharts.
